# 🌟 Элементум — карточная игра с NFC, PvP и экономикой

> Эпическая карточная RPG с пошаговыми боями, сюжетом на 25 глав, коллекцией из 60+ карт, ежедневными наградами, квестами, достижениями, PvP-ареной, системой друзей, торговлей, крафтом и NFC-сканированием.  
> Проект написан на **Python + Flask + SQLite**.

---

## ✨ Особенности

- 📖 **Сюжет** с уникальными текстами, боссами и наградами (карты + монеты).
- ⚔️ **Боевая система** с преимуществом стихий, щитами, пропуском хода и рекурсивной картой «Цепная молния».
- 🃏 **Одноразовые карты** – каждое использование расходует карту.
- 🎁 **Ежедневный кейс** с редкостями и стриком.
- 🏆 **Ежедневные квесты** и достижения (рекурсивные цепочки).
- 👤 **Профиль** с аватаркой (загрузка изображений), витриной любимых карт, сменой ника и стихии (раз в месяц).
- 📡 **NFC-сканер** (Web NFC API) – получение редких карт и усиление в бою (раз в 3 дня).
- ⚔️ **PvP-режим** с рейтингом (ELO), очередью поиска соперника, таблицей лидеров.
- 👥 **Друзья** – добавление по ссылке-приглашению, подтверждение заявок, просмотр чужих профилей.
- 💱 **Торговля** картами (карта → карта или карта → монеты) с активными предложениями.
- 🛒 **Магазин** – публичный (информация о товарах) и игровой (покупка карт за монеты).
- 🔨 **Крафт** – 3 карты одной редкости → 1 карта следующей редкости.
- 🪙 **Монеты** – внутренняя валюта (начисляются за победы в боях, ежедневный бонус, завершение сюжета). Тратятся на обучение стихии, покупку карт в магазине.
- 🎓 **Обучение стихии** – стоит 100 монет, увеличивает множитель урона на +0.2.
- 🚫 **Отключение вызовов** – игрок может запретить вызовы на дуэль.

## 🚀 Установка и запуск

```bash
git clone <repository>
cd game
pip install -r requirements.txt
python app.py
Откройте в браузере: https://localhost:5000 (для NFC нужен HTTPS). 
```

Работа с NFC (требуется HTTPS)
NFC-сканер доступен только через HTTPS. В режиме разработки можно использовать самоподписанный сертификат.

Сгенерируйте сертификат и ключ (если ещё нет):

```bash
openssl req -x509 -newkey rsa:4096 -keyout key.pem -out cert.pem -days 365 -nodes 
```
В app.py укажите пути к файлам:

```python
ssl_context=('cert.pem', 'key.pem')
```

Запустите сервер:

```bash
python app.py
```

Переменные окружения (.env)
```ini
SECRET_KEY=your-secret-key-here
MAIL_SERVER=smtp.yandex.ru
MAIL_PORT=465
MAIL_USE_SSL=True
MAIL_USERNAME=your_email@yandex.ru
MAIL_PASSWORD=your_app_password
MAIL_DEFAULT_SENDER=your_email@yandex.ru
YOOKASSA_SHOP_ID=your_shop_id
YOOKASSA_SECRET_KEY=your_secret_key
```

## 📡 NFC-механика
Получение карты – сканирование NFC-метки с записанным nfc_hash выдаёт соответствующую карту (раз в 3 дня).
Усиление в бою – во время своего хода можно отсканировать любую NFC-метку, связанную с картой, и применить усиленную версию (урон/щит/лечение ×1.5) без расхода карты из колоды. Один раз в бой и раз в 3 дня.

```text

```

## 🧪 Тестирование
Регистрация/логин – работает через email или никнейм.

Сюжет – последовательные бои с врагами (AI).

PvP – очередь, матчмейкинг, рейтинг ELO, полная боевая механика.

Торговля – создание/принятие предложений с выбором типа желаемого (карта или монеты).

Крафт – 3 карты одной редкости → 1 случайная карта выше на ступень.

Магазин – игровой магазин за монеты (ассортимент обновляется при каждом заходе).

Друзья – приглашение по ссылке, заявки, удаление.

NFC – работает только в Chrome на Android через HTTPS.
